Getting Started

The 1,2,3 of getting rockin' with Sensei!

Start Zookeeper

Zookeeper manages Sensei's network topology or cluster.

Start your ZooKeeper instance: (detailed instruction here) 

$ZK_HOME/bin/zkServer.sh start

Create & configure an app

Putting together some configuration. (Example: Cars)

Details »

Start your Sensei node by pointing to your configuration directory (the following example starts the car example application): 

$SENSEI_HOME/bin/start-sensei-node.sh conf

Voila! That's it!

You can now interact with your Sensei node with either the Sensei Web-console or the Sensei Shell!


Web-console

Sensei comes with a web-console (written using Sensei's Javascript API) to help you interact with your Sensei cluster.

Point your favorite browser to: http://localhost:8080

Learn details of the Javascript API here.

Shell

You can also query your Sensei cluster with BQL using the Sensei Shell.

Start the Sensei Shell: host/port information optional, defaults to localhost:8080 

$SENSEI_HOME/bin/sensei-shell.sh
