Getting the code

Sensei is hosted on Github using the Git version control system. You can get the code with: 

git clone git://github.com/senseidb/sensei.git

Building the code

SenseiDB uses Apache Maven for building.

Go into top-level directory for Sensei and do: 

ant
or 
mvn package

Testing

First start Zookeeper by going to Zookeeper install directory and do: 

./bin/zkServer.sh start

You can run the unit test with the following command: (Make sure Zookeeper is running): 

mvn test
