Usage

Some usages of Sensei

Sensei is powering various parts of the LinkedIn site:

  • Homepage
  • Signal
  • Search properties, e.g. people,company,jobs etc. (built on Sensei core, e.g. Bobo/Zoie etc.)

Kormox is using Sensei DB to power the core of its data processing engine. Kormox technology oversees and understands all unstructured data in an organization. It makes it really easy to classify information and make it truly manageable along Security, Knowledge Management, Productivity and Business Intelligence.

Kormox has been utilizing Sensei DB since early 2010 and has developed a wide array of extensions, enabling real-time distributed relational join-like functionality (within Sensei DB and against external sources) as well as the ability to update record fields directly in-store.
