Java API

The Java API is a wrapper around the Rest API.

Example: 

   SenseiClientRequest senseiRequest  =  SenseiClientRequest.Builder.paging(0,10).
                                         .addSelection(Selection.terms("color","red"))
                                         .addSelection(Selection.range("price","*","2000"))
                                         .addSort(Sort.desc("color"))
                                         .addFacet("category",Facet.builder().max(5).build())
                                         .query(Queries.stringQuery("cool"))
                                         .build();
                                           
   SenseiResult senseiResult = new SenseiServiceProxy().sendRequest("http://localhost:8079/sensei/", senseiRequest);
